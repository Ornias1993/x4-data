well you dont need 3d studio max, as you can use gmax either. However im not sure if blender can do everything you need - maybe it can? To use blender it will need to export to bod through dbox2 (can blender use plug-ins from 3dmax/gmax?). I guess if it cant, then you could always have gmax with dbox2, then with blender export to a file-type gmax understands, then export the model in gmax (through dbox2).

as for the programs you need from double shadow:

- dbox2 (a plug-in for gmax/3ds so you can load/export bod/scenes),
- X2 BOD Compiler (to compile/compress/decompile/decompress bob/bod files),
- Mod Manager (to build the zip like archive for your mod),
- X3 Editor (to edit types files, so you can add your ship to like types/tships),
