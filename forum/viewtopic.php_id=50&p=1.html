Hopefully it should work this time:

Download the zip file and extract the following files into your <game>/X2/mods folder:
(ie. C:/Program files/X2/mods):
- Rogueys_Extreme v2.dat
- Rogueys_Extreme v2.cat

After that then run the game. On the first screen where it says about setting the resolution, input device, player logo etc. Click on "Select mod package", you should then have list of mods installed, select one then click on "Ok". Once that is done you should be sent back to the first screen with the setup option. Just play the game as normal. You only need to select the mod package once.
